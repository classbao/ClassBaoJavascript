1，根据索引移除数组中指定项：

        
var myArray = [1, 2, 3];
var myNewArray = CBJS.Array.removeByIndex(myArray, 2);
console.log("removeByIndex", myNewArray);
//output: [1, 2]

参数解析：CBJS.Array.removeByIndex(array, index)

> array:原数组

> index:数组下标索引（从0开始）

> 返回值:移除指定项后的数组

项目值
根据索引移除数组中指定项

2，移除数组中指定项：

        
var myArray = [1, 2, 3];
var myNewArray = CBJS.Array.removeByItem(myArray, 2);
console.log("removeByItem", myNewArray);
//output: [1, 3]

参数解析：CBJS.Array.removeByItem(array, removeItem)

> array:原数组

> removeItem:原数组中的元素

> 返回值:移除指定元素后的数组

项目值
移除数组中指定项

3，正则表达式分割字符串为数组：

        
var regexSplit = CBJS.RegexSplit("我爱中国我爱电影我爱编程", /(我爱)/ig);
console.log("RegexSplit", regexSplit);
//output: ["中国", "电影", "编程"]

参数解析：CBJS.RegexSplit(splitString, regexFormat)

> splitString:要分割的字符串

> regexFormat:正则表达式

> 返回值:数组

项目值
正则表达式分割字符串为数组
